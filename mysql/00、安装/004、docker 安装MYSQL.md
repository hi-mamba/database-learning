# docker 安装MYSQL
```shell script
shell> docker pull mysql/mysql-server:tag
```
  Refer to the list of supported tags above. 
  If :tag is omitted, the latest tag is used, 
  and the image for the latest GA version of MySQL Server is downloaded.
